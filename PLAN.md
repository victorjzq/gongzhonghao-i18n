## 战略：Medium SEO 抢位 + Substack 订阅沉淀

### Phase 1: Medium 精品首发（本周）
目标：用 Medium DA95 抢 Google 长尾词排名

1. 从 351 篇中精选 20 篇「入门级 + 高搜索量」文章
2. SEO 优化标题（加 "Al Brooks" 品牌词 + 长尾词）
3. 每天发 2-3 篇（不批量灌），加 canonical tag
4. Tags: price-action, trading, al-brooks, technical-analysis, day-trading

选文优先级：
- 交易书籍系列（12篇）→ 书评类搜索量大
- 入门与成长（13篇）→ 新手入口
- 交易原则（30篇）→ 核心方法论
- 交易手册（17篇）→ 长文深度内容

### Phase 2: Substack 系列化（下周）
目标：建立订阅者列表

1. 创建 "Price Action Notes" newsletter
2. 按主题整理成 7 个系列（趋势/区间/突破/信号/心理/策略/书评）
3. 免费内容引流，深度解读设为付费
4. 每周 2-3 篇节奏

### Phase 3: Reddit 社区种草（持续）
目标：从交易社区引流到 Medium/Substack

- r/daytrading: 发图表分析帖，签名引流

### Phase 4: 多语言扩展（验证 PMF 后）
- 先观察英文 Medium 流量，目标：单篇 500+ views
- 达标后启动越南语（vi 已翻译 351 篇）发 Medium

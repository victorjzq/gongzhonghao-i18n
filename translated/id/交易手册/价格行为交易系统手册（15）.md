#### **Strategi Trading Breakout**

Identifikasi bar breakout

    Bar tren dengan body besar

Tunggu follow-through

    Jika bar berikutnya juga merupakan bar tren yang kuat ke arah yang sama, probabilitas breakout yang sukses meningkat secara substansial

Entry

    Buy The Close

Breakout Pullback — tunggu pullback pertama setelah breakout untuk entry

    Retest titik breakout, retest moving average, konsolidasi minor

Membedakan breakout sejati vs. palsu (sistem penilaian 10+2)

Karakteristik breakout sejati (skor tinggi):

    ┌────────────────┬───────────────────┬───────┐
    │    Kriteria    │     Standar       │ Skor  │
    ├────────────────┼───────────────────┼───────┤
    │ Kontinuitas bar│ Bar besar         │ +10%  │
    │                │ berturut-turut    │       │
    │ Harga close    │ Dekat ekstrem     │ +10%  │
    │ Micro gap      │ Tidak ada overlap │ +10%  │
    │                │ antar bar         │       │
    │ Tick gap       │ Gap di open       │ +10%  │
    │ Progres H/L    │ Secara progresif  │ +10%  │
    │                │ naik/turun        │       │
    │ Tanpa shadow   │ Atau sangat       │ +10%  │
    │                │ pendek            │       │
    │ Tanpa doji     │ Semua bar tren    │ +10%  │
    │ Tanpa bar      │ Pergerakan satu   │ +10%  │
    │ reversal       │ arah              │       │
    │ Urgensi opening│ Gap up/down open  │ +10%  │
    │ Jarak breakout │ Jarak yang        │ +10%  │
    │                │ ternotasi         │       │
    │ Durasi waktu   │ Minimal 3 bar     │ +20%  │
    └────────────────┴───────────────────┴───────┘

Total skor >85% = breakout berkualitas tinggi

Karakteristik breakout palsu (skor rendah):

    ❌ Langsung diikuti oleh bar reversal yang kuat

Breakout gagal

    ❌ Langsung diikuti oleh bar reversal yang kuat

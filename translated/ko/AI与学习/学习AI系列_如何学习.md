AI 학습 시리즈: 어떻게 학습할 것인가
===========

**이것은 AI 학습에 관한 나의 네 번째 글입니다**
-----------------



방법론 매뉴얼



1. 문제 정의

1.1 학습이 해결해야 할 핵심 문제

환경 속에서 생존하기 위한 생명의 핵심 능력은 "정보를 예측하는 능력"이다 — 이익을 추구하고 해로움을 피하며, 환경 속에서 자신의 상태를 주체적으로 유지하는 것이다.

예측은 두 가지로 분류된다:

\- 경험적 예측: 이미 알고 있는 것만 예측할 수 있고, 모르는 것은 예측할 수 없다

\- 모델 예측: 알려진 상황에서 미지의 상황을 예측할 수 있는 규칙을 찾아낸다

학습이 해결해야 할 문제 = 일반화 능력을 획득하여, 한 번도 만나본 적 없는 상황을 처리하는 것이다.

1.2 학습 실패의 근본 원인

1. 목표 오류: "잘못된 목표는 직접적으로 잘못된 결과를 초래한다" — "일반화해야 할 내용"을 "재현해야 할 내용"으로 취급한다

2. 기언대학(말의 기억으로 학습을 대체): "'지식의 언어적 재료'를 '지식' 자체로 오인하고, '언어 재료의 기억'을 '학습'으로 오인한다"

3. 감각에 의존한 학습: "학습 과정에서 자신의 목표가 무엇인지, 어떻게 목표를 달성할 것인지 명확하지 않다"



2. 판단 순서 (핵심)

2.1 첫 번째 판단: 재현인가 일반화인가

"습득해야 할 내용이 있을 때, 먼저 그것이 재현해야 할 내용인지, 일반화해야 할 내용인지를 판단하여, 학습 방법을 채택할지 기억 방법을 채택할지를 결정한다."

입력: 습득해야 할 내용

판단: 해당 내용이 한 번도 만나본 적 없는 상황을 처리해야 하는가?

\-- 아니오 -> 재현 목표 -> 기억 방법

\-- 예 -> 일반화 목표 -> 학습 방법

2.2 재현 경로

"재현해야 할 내용이라면, 눈앞의 자료를 외우면 된다."

실행: 자료를 직접 기억한다



2.3 일반화 경로

"일반화해야 할 내용이라면, 눈앞의 자료는 기억하기 위한 것이 아니라, 머릿속에 매핑을 구축하여 무한한 새로운 상황에 대처하기 위한 것이다."

단계 1: 입력 공간과 출력 공간을 명확히 한다

"어떤 지식을 학습할 때든, 먼저 해당 지식의 입력 공간과 출력 공간을 명확히 해야 한다."

\- 입력 공간: 지식의 적용 범위 = 입력 변수의 값 범위

\- 출력 공간: 예측 목표 = 이 지식이 우리에게 어떤 정보를 알려줄 수 있는가

단계 2: 자료를 수집한다

"'구체적 사례'와 '타인의 서술'을 수집한다."

자료는 두 가지로 분류된다:

\- 구체적 사례 -> 대상층에 봉사한다

\- 타인의 서술 -> 공상층에 봉사한다

단계 3: 정리와 편성

"'구상 수준의 서술'과 '추상 수준의 서술'을 분류하여, '판별 모델'과 '연결 모델' 구축에 명확한 입력과 출력을 제공한다."

단계 4: 모델을 구축한다

"'하상 구조'를 세우고, '대상과 개념 사이의 관계'를 정리하여 '판별 모델'을 구축한 다음, '서로 다른 개념 간의 관계'를 정리하여 '연결 모델'을 구축한다."

실행 순서:

1\. 대상층 (구체적 사례) -> 다른 사례들을 비교 -> 공유 속성을 찾아낸다

2\. 공상층 (추상적 규칙) -> 범용 판별 규칙을 구축한다

3\. 연결층 -> 서로 다른 개념 간의 관계를 정리한다

단계 5: 효과를 검증한다

"새로운 상황을 사용하여 자신이 구축한 '범용 판별 규칙'의 판별 능력을 검증하고, 그것이 충분히 범용적인지 확인한다."

판단 기준:

\- 판단이 맞음 -> "정말로 학습하고 있다... 정말로 '범용 판별 규칙'을 구축하고 있다"

\- 판단이 틀림 -> 역추적에 진입한다

2.4 조건 변화: 역추적 트리거

"우리는 무수히 많은 새로운 상황을 만날 것이고, 여전히 판단을 잘못할 수 있다... 이는 머릿속에 구축한 '부호화'에 관한 '범용 판별 규칙'이 그다지 범용적이지 않다는 것을 의미한다. 이때, 이전 단계로 돌아가 새로운 학습 자료를 다시 수집하고, '범용 판별 규칙'을 재구축해야 한다."

검증 실패

\-- 단계 2로 역추적 (자료 수집)

\-- 단계 3-5를 재실행

3. 핵심 개념



| 용어 | 조작적 정의 |
|------|-----------|
| 재현 | 경험을 있는 그대로 재현하여, 알려진 상황을 예측한다 |
| 일반화 | 알려진 것으로부터 규칙을 구축하여, 보지 못한 상황을 해결한다 |
| 입력 공간 | 지식의 적용 범위 (정의역) |
| 출력 공간 | 지식의 예측 목표 (치역) |
| 대상층 | 하층, 구체적으로 관찰 가능한 현상 |
| 공상층 | 상층, 추상적 규칙 |



4. 사용 전제

1. 첫 번째 판단을 먼저 완료해야 한다

"목표는 우리가 가장 먼저 확인해야 할 것이다. 왜냐하면 잘못된 목표는 직접적으로 잘못된 결과를 초래하기 때문이다."

2. 입력과 출력을 먼저 명확히 해야 한다

"어떤 지식을 학습할 때든, 먼저 해당 지식의 입력 공간과 출력 공간을 명확히 해야 한다. 줄여서 '입출력 명확화'라 한다."

3. 자료의 용도는 목표 유형에 따라 결정된다

\- 재현 목표 -> 자료는 기억에 사용된다

\- 일반화 목표 -> 자료는 매핑 구축에 사용된다



5. 실패 조건 / 오용 위험

5.1 기언대학

\- 표현 양상: "말은 있으나 의미는 비어 있다" — 언어 세계에서 새로운 기호만 만들 뿐, 의미 세계에서 대응하는 개념을 구축하지 않는다

\- 판단 기준:

\- 이름은 알지만 사람을 알아보지 못한다: "'엔트로피'라는 개념의 이름만 기억했을 뿐... '엔트로피'의 개념을 구축하지 못했다"

\- 교과서대로 암송한다: "지식의 서술만 기억했을 뿐... 서술을 바꾸면 알아보지 못한다"

5.2 지식 붕괴

하층 상실 (대상층 상실)

\- 표현 양상: "일종의 '공중에 뜬 경험'이 생긴다. '타인과의 대화에서' 또는 '필기시험이나 구술시험에서'만 만날 수 있는 특수한 경험"

\- 판단 기준: "시험이나 대화 소재로만 사용할 수 있을 뿐, 실제 문제를 해결하는 능력으로 전환할 수 없다"

상층 상실 (공상층 상실)

\- 표현 양상: "'완전히 동일한 상황'만 해결할 수 있고, 상황이 바뀌면 어떻게 해결해야 할지 모른다"

\- 판단 기준: "'보지 못한 상황'을 해결할 수 없다, 하나를 알면 열을 알 수 없다"

상하층 불일치

\- 표현 양상: "학습자가 '판별 모델'을 가지고 있지만, 이 '판별 모델'이 맞지 않아 '하층 대상'이 '적합하지 않은 상층 개념'으로 판별된다"

\- 판단 기준: "지식을 운용할 때 추론 오류가 발생한다"

5.3 기계적 기억

\- 발동 조건: "일반화에 사용해야 할 학습 자료를 암기에 사용한 경우"

\- 판단 기준: 일반화해야 할 내용(법칙, 정리)을 기계적으로 암기한다

5.4 과적합 (Overfitting)

\- 정의: "학습 시 본 상황만 기억했을 뿐, 보지 못한 상황을 해결할 수 없다"

\- 판단 기준: "타인이 알려준 상황만 해결할 수 있고, 알려주지 않은 상황은 해결할 수 없다"

5.5 유형과 방법의 불일치

"일반화에 사용해야 할 학습 자료를 암기에 사용하면 기계적 기억의 함정에 빠진다. 재현에 사용해야 할 기억 자료를 일반화에 사용하면 불필요한 일이 되어 효율이 떨어진다."



6. 체계적 평가

6.1 적용 범위

"모든 유형의 지식 학습에 대응할 수 있다. '연구 수행', '운전 학습', '요리 학습', '게임 학습'이라 하더라도 모두 대응하는 표준화된 단계가 있다."

6.2 적용 불가 상황

"당신의 주민등록번호처럼, 아무리 일반화해도 여전히 하나의 주민등록번호일 뿐이다. 이 경우 채택해야 할 것은 기억 방법이지, 학습 방법이 아니다."

6.3 검증 성공 기준

"판단이 정확하다면, 그것은 당신이 정말로 학습하고 있다는 것을 의미한다... 정말로 '범용 판별 규칙'을 구축하고 있는 것이지, 단순히 '학습 자료'를 기억하고 있는 것이 아니다."

6.4 검증 실패 기준

"'타인의 서술을 기억할 수 있는지 여부'를 '학습 효과'를 검증하는 방식으로 잘못 인식한다."


단순한 문제 vs. 복잡한 문제: "단순한 문제만 해결한다면, 사람들은 이론이 필요하지 않다. 이론의 강점은 복잡한 문제를 분석하고 해결하는 것을 도울 수 있다는 데 있다."

암묵지의 언어적 한계: "'암묵지'의 학습에 있어, 언어의 역할은 더욱 제한적이다. 사람들은 언어를 사용하여 '암묵지'를 정확하게 서술할 수 없다"

Serie de Aprendizaje de IA: Cómo Aprender
===========

**Este es mi 4.o artículo sobre el aprendizaje de IA**
-----------------



Manual de Metodología



I. Definición del Problema

1.1 El problema central que el aprendizaje debe resolver

La capacidad central de la vida para sobrevivir en un entorno es "la capacidad de predecir información" — buscar beneficios y evitar daños, mantener activamente el propio estado dentro del entorno.

La predicción se divide en dos categorías:

\- Predicción basada en experiencia: Solo puede predecir lo conocido, no lo desconocido

\- Predicción basada en modelos: Encontrar patrones a partir de situaciones conocidas que puedan predecir situaciones desconocidas

El problema que el aprendizaje debe resolver = adquirir la capacidad de generalización, para manejar situaciones nunca antes encontradas.

1.2 Las causas raíz del fracaso en el aprendizaje

1. Objetivo incorrecto: "Un objetivo incorrecto conducirá directamente a resultados incorrectos" — tratar "contenido que necesita ser generalizado" como "contenido que necesita ser reproducido"

2. Sustituir memorización por aprendizaje: "Confundir 'el material lingüístico del conocimiento' con 'el conocimiento' en sí, confundir 'la memorización del material lingüístico' con 'aprendizaje'"

3. Aprender por intuición: "No tener claro cuál es el objetivo propio durante el proceso de aprendizaje, ni cómo lograrlo"



II. Secuencia de Decisión (Núcleo)

2.1 Primera Decisión: Reproducción o Generalización

"Cuando se tiene un contenido que necesita ser dominado, primero determina si es un contenido que necesita ser reproducido, o un contenido que necesita ser generalizado, y usa esto para decidir si adoptar un método de aprendizaje o un método de memorización."

Entrada: Contenido que necesita ser dominado

Decisión: ¿Este contenido necesita manejar situaciones nunca antes encontradas?

\-- No -> Objetivo de reproducción -> Método de memorización

\-- Sí -> Objetivo de generalización -> Método de aprendizaje

2.2 Ruta de Reproducción

"Si el contenido necesita ser reproducido, entonces simplemente memoriza el material que tienes delante."

Ejecución: Memorizar directamente el material



2.3 Ruta de Generalización

"Si el contenido necesita ser generalizado, entonces el material que tienes delante no es para memorizar, sino para ayudarte a construir un mapeo en tu mente para manejar infinitas situaciones nuevas."

Paso 1: Aclarar el espacio de entrada y el espacio de salida

"Al aprender cualquier conocimiento, primero debes aclarar el espacio de entrada y el espacio de salida de ese conocimiento."

\- Espacio de entrada: El alcance de aplicación del conocimiento = el rango de valores de las variables de entrada

\- Espacio de salida: El objetivo de predicción = qué información puede decirnos este conocimiento

Paso 2: Recopilar materiales

"Recopilar 'ejemplos concretos' y 'descripciones de otros'."

Los materiales se dividen en dos categorías:

\- Ejemplos concretos -> Sirven a la capa de objetos

\- Descripciones de otros -> Sirven a la capa de abstracción

Paso 3: Organizar y ordenar

"Categorizar las 'descripciones de nivel concreto' y las 'descripciones de nivel abstracto' por separado, proporcionando entradas y salidas claras para construir 'modelos discriminativos' y 'modelos asociativos'."

Paso 4: Construir modelos

"Construir una 'estructura de abajo hacia arriba', intentar aclarar 'la relación entre objetos y conceptos' para construir 'modelos discriminativos', luego intentar aclarar 'las relaciones entre diferentes conceptos' para construir 'modelos asociativos'."

Secuencia de ejecución:

1\. Capa de objetos (ejemplos concretos) -> Comparar diferentes ejemplos -> Encontrar atributos compartidos

2\. Capa de abstracción (reglas abstractas) -> Construir reglas discriminativas universales

3\. Capa de asociación -> Aclarar las relaciones entre diferentes conceptos

Paso 5: Verificar la efectividad

"Usar situaciones nuevas para verificar la capacidad discriminativa de las 'reglas discriminativas universales' que has construido, y ver si son suficientemente universales."

Criterios de evaluación:

\- Juicio correcto -> "Verdaderamente aprendiendo... verdaderamente construyendo 'reglas discriminativas universales'"

\- Juicio incorrecto -> Entrar en retroceso

2.4 Cambio de Condición: Activación de Retroceso

"Nos encontraremos con innumerables situaciones nuevas y es posible que aún hagamos juicios incorrectos... Esto significa que las 'reglas discriminativas universales' que has construido en tu mente sobre 'codificación' no son tan universales. En este punto, necesitas volver a los pasos anteriores, recopilar nuevos materiales de aprendizaje y reconstruir las 'reglas discriminativas universales'."

Verificación fallida

\-- Retroceder al Paso 2 (Recopilar materiales)

\-- Re-ejecutar Pasos 3-5

III. Conceptos Centrales



| Término | Definición Operacional |
|---------|----------------------|
| Reproducción | Reproducir la experiencia tal cual, predecir situaciones conocidas |
| Generalización | Construir patrones a partir de lo conocido para resolver situaciones no vistas |
| Espacio de Entrada | El alcance de aplicación del conocimiento (dominio) |
| Espacio de Salida | El objetivo de predicción del conocimiento (codominio) |
| Capa de Objetos | Capa inferior, fenómenos concretos observables |
| Capa de Abstracción | Capa superior, patrones abstractos |



IV. Prerrequisitos de Uso

1. La primera decisión debe completarse primero

"El objetivo es lo primero que debemos examinar, porque un objetivo incorrecto conducirá directamente a resultados incorrectos."

2. La entrada y salida deben aclararse primero

"Al aprender cualquier conocimiento, primero debes aclarar el espacio de entrada y el espacio de salida de ese conocimiento, abreviado como 'aclarar entrada y salida'."

3. El uso de los materiales depende del tipo de objetivo

\- Objetivo de reproducción -> Los materiales se usan para memorización

\- Objetivo de generalización -> Los materiales se usan para construir mapeos



V. Condiciones de Fracaso / Riesgos de Uso Incorrecto

5.1 Sustituir Memorización por Aprendizaje

\- Manifestación: "Las palabras existen pero el significado está vacío" — solo crear nuevos símbolos en el mundo lingüístico, sin construir los conceptos correspondientes en el mundo del significado

\- Criterios de evaluación:

\- Conocer el nombre pero no reconocer a la persona: "Solo memorizó el nombre del concepto 'entropía'... pero no construyó el concepto de 'entropía'"

\- Recitar de memoria: "Solo memorizó la descripción del conocimiento... cámbiala y ya no la reconoce"

5.2 Colapso del Conocimiento

Pérdida de la capa inferior (Pérdida de la capa de objetos)

\- Manifestación: "Produce una especie de 'experiencia suspendida', una experiencia especial que solo puede encontrarse 'en conversaciones con otros' o 'en exámenes escritos u orales'"

\- Criterios de evaluación: "Solo puede usarse para exámenes o como tema de conversación, no puede transformarse en la capacidad de resolver problemas prácticos"

Pérdida de la capa superior (Pérdida de la capa de abstracción)

\- Manifestación: "Solo puede resolver 'situaciones exactamente idénticas', no sabe cómo resolver cuando la situación cambia"

\- Criterios de evaluación: "No puede resolver 'situaciones no vistas', no puede extrapolar de un ejemplo a muchos"

Desajuste entre capas

\- Manifestación: "El aprendiz tiene un 'modelo discriminativo', pero este 'modelo discriminativo' no coincide, causando que 'los objetos de la capa inferior' sean clasificados en 'conceptos de la capa superior inadecuados'"

\- Criterios de evaluación: "Produce falacias de razonamiento al aplicar el conocimiento"

5.3 Memorización Mecánica

\- Condición de activación: "Si los materiales de aprendizaje que deberían usarse para generalización se usan para memorización"

\- Criterios de evaluación: Memorizar mecánicamente contenido que necesita ser generalizado (leyes, teoremas)

5.4 Sobreajuste (Overfitting)

\- Definición: "Solo memorizó las situaciones vistas durante el aprendizaje, pero no puede resolver situaciones no vistas"

\- Criterios de evaluación: "Solo puede resolver situaciones que otros le han contado, pero no puede resolver situaciones de las que no ha sido informado"

5.5 Desajuste de Tipo y Método

"Si los materiales de aprendizaje que deberían usarse para generalización se usan para memorización, se caerá en la trampa de la memorización mecánica. Si los materiales de memorización que deberían usarse para reproducción se usan para generalización, será innecesario y conducirá a baja eficiencia."



VI. Evaluación Sistemática

6.1 Alcance de Aplicación

"Puede manejar el aprendizaje de todo tipo de conocimiento, incluso 'hacer investigación', 'aprender a conducir', 'aprender a cocinar', 'aprender a jugar videojuegos' — todos tienen pasos estandarizados correspondientes."

6.2 Situaciones No Aplicables

"Como tu número de identificación — por mucho que intentes generalizarlo, sigue siendo solo un número de identificación. En este caso, debes usar un método de memorización, no un método de aprendizaje."

6.3 Criterios de Verificación Exitosa

"Si tu juicio es correcto, significa que verdaderamente estás aprendiendo... verdaderamente construyendo 'reglas discriminativas universales', en lugar de simplemente memorizar 'materiales de aprendizaje'."

6.4 Criterios de Verificación Fallida

"Confundir erróneamente 'si puedes recordar las descripciones de otros' como la forma de verificar la 'efectividad del aprendizaje'."


Problemas simples vs. complejos: "Si solo se resuelven problemas simples, las personas no necesitan teoría. El poder de la teoría reside en su capacidad para ayudarnos a analizar y resolver problemas complejos."

Las limitaciones lingüísticas del conocimiento tácito: "Para el aprendizaje del 'conocimiento tácito', el papel del lenguaje es aún más limitado — las personas no pueden usar el lenguaje para describir con precisión el 'conocimiento tácito'"

交易是数学游戏。机构算法会根据几何对称性寻找目标位。成功的交易来自于正确识别市场环境并执行与之匹配的策略。

为了约束交易幻觉，避免提前猜想，我选择先看信号K的方向，再检查三要素来评估交易质量，决定是否交易。

  

三要素指的是：背景，形态，位置。

  

背景是最复杂的要素，是抽象的概念，需要其他概念来约束。

  

市场环境

当前市场处于多头环境、空头环境还是中性环境？

  

最简单有效的方法是观察价格与指数移动平均线（EMA）的关系。价格在均线之上且均线向上倾斜，为多头环境。

  

市场行为

当前是趋势市场还是交易区间？

特征

表现

均线状态

EMA 明显倾斜

回调深度

回调浅且短暂

价格结构

持续创新高/新低

缺口

缺口保持不回补

  

微通道（Microchannel）是强趋势的标志：

“微通道是指一系列没有回调的 K 线。多头微通道：每根 K 线的低点都等于或高于前一根 K 线的低点。”

判断Always In 方向
--------------

Always In 意味着如果你必须此刻入场且必须持有一个方向的仓位，你会选择哪个方向。它帮助交易者过滤掉逆势操作的冲动。这是一个强制二选一的思维工具，移除了"观望/中立"的选项。

  

确定 Always In 的主要目的是为了过滤错误的交易方向。如果市场处于 Always In Long 状态，那么你就应该把’做空’这个选项从决策树中移除，只寻找买入机会。同时可以消除犹豫，当Always In Long时，即使"感觉太高"也应该买入，也是持仓依据：只要状态不变，就继续持有。

  

Always In = 判断当前是多头还是空头更容易赚钱，然后只跟随那一方交易。

  状态: Always In Long

  含义: 多头控制市场，做多更容易赚钱

  特征: 连续大阳线、突破创新高、空头反扑被打压

  ────────────────────────────

  状态: Always In Short

  含义: 空头控制市场，做空更容易赚钱

  特征: 连续大阴线、跌破支撑、多头反弹失败

  简单判断方法，问自己："现在谁更容易赚钱？"

  多头买入就能赚 → Always In Long → 只做多

  空头卖出就能赚 → Always In Short → 只做空

  两边都能赚/都不好赚 → 方向不明确 → 等待

  

Always In 判断方法（具体执行标准）

  第一步：看K线特征

  ┌────────────┬────────────────────────┬────────────────────────┐

  │    特征    │     Always In Long     │    Always In Short     │

  ├────────────┼────────────────────────┼────────────────────────┤

  │ K线实体    │ 大阳线，收盘接近最高价 │ 大阴线，收盘接近最低价 │

  ├────────────┼────────────────────────┼────────────────────────┤

  │ K线重叠    │ 连续K线之间重叠少      │ 连续K线之间重叠少      │

  ├────────────┼────────────────────────┼────────────────────────┤

  │ 与均线关系 │ 收盘在20EMA之上        │ 收盘在20EMA之下        │

  ├────────────┼────────────────────────┼────────────────────────┤

  │ 缺口       │ 低点高于前K线高点      │ 高点低于前K线低点      │

  └────────────┴────────────────────────┴─────────────────

  第二步：需要几根K线确认？

  ┌──────────┬─────────────────────────────────────────────┐

  │   情况   │                  所需K线数                  │

  ├──────────┼─────────────────────────────────────────────┤

  │ 标准确认 │ 2根：突破K线 + 跟随K线（都是同向强趋势K线） │

  ├──────────┼─────────────────────────────────────────────┤

  │ 稳妥确认 │ 3根：连续3根同向趋势K线                     │

  ├──────────┼─────────────────────────────────────────────┤

  │ 高度确认 │ 4根：连续4根同向K线且收盘在均线一侧         │

  ├──────────┼─────────────────────────────────────────────┤

  │ 极端情况 │ 1根：巨大的惊喜棒（但建议仍等跟随确认）     │

  └──────────┴─────────────────────────────────────────────┘

  第三步：检查清单

  判断Always In前，逐项检查：

  □ 1. 有无大实体趋势K线？（收盘接近极值）

  □ 2. 下一根K线是否同向？（不是十字星或反向K线）

  □ 3. K线是否收盘在20EMA的一侧？

  □ 4. 是否突破了关键点位？（区间边界/前高低点）

  □ 5. 逆势者是否被套？（没有回调让他们保本离场）

 5项中满足4项以上 → 可确认Always In

 具体K线组合示例

 Always In Long（做多）

示例1：突破+跟随

  K线1: 大阳线，收盘接近最高价

  K线2: 又一根阳线，收盘高于K线1高点

  → 确认 Always In Long

示例2：连续3阳

  K线1: 阳线

  K线2: 阳线，收盘更高

  K线3: 阳线，收盘更高

  → 确认 Always In Long

 示例3：突破均线

  K线1: 大阳线突破并收盘在20EMA之上

  K线2: 阳线，维持在均线上方

  → 确认 Always In Long

Always In Short（做空）

示例1：连续阴线

  K线1: 大阴线

  K线2: 大阴线

  K线3: 阴线，收盘在20EMA下方

  → 确认 Always In Short

示例2：反转失败

  K线1: 阳线（多头尝试）

  K线2: 更大的阴线吞没阳线

  → 确认 Always In Short

示例3：突破+跟随

  K线1: 大阴线突破区间下沿

  K线2: 又一根阴线

  → 确认 Always In Short

简化判断流程，看到一根大趋势K线，等待下一根K线

      ├── 同向强K线 → 确认Always In → 只做这个方向

      │

      └── 十字星/反向K线 → 不确认 → 等待或视为震荡

  

一句话总结，大趋势K线 + 同向跟随K线 + 收盘在均线一侧 = Always In确立

  

什么情况下 Always In 方向会改变？

  

1. 反向出现强趋势 K 线

2. 连续的反向 K 线收盘

3. 封闭之前的缺口

4. 突破趋势线后测试极值失败

  

如果在 Always In Long 的市场体制下错误地判断为可以做空，通常只能获得极小的剥头皮利润，或者直接面临亏损。

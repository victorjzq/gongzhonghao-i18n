价格行为学：支撑与阻力

控制价格的，是支撑与阻力，而不是新闻

支撑是市场下方的一个价格带──买方可能比卖方更强的位置；阻力则相反。行情在这些“拐点”（inflection point）要么突破、要么反转，因此总潜伏一次大幅波动的潜力。止损和仓位必须贴合你所交易的时间周期来设定。

一、趋势阶段：80 % 规则  
强空头趋势  
80 % 的支撑都会被击穿，80 % 的反转尝试都会失败。价格不会在前低形成大量买盘，而是不断下行；多次阳线、小型双底的反转也常告失败。  
强多头趋势  
80 % 的阻力都会被突破，市场具有惯性，倾向延续原向。

二、交易区间：同样的 80 % 规则  
当市场进入震荡区，80 % 的趋势突破尝试会失败。  
区间顶部＝明显阻力，底部＝明显支撑。交易者预期空头突破会失败，倾向低买高卖、快速剥头皮，因为下跌与上涨幅度都被限制。

三、支撑阻力的吸引力与测试  
市场不断试探支撑与阻力，寻找公允价格带。这才是交易者最值得研究的核心，而非新闻。  
观察图表左侧是否已经测试过某个结构：若尚未测试，行情通常先到价位再决定去留；若已多次测试，随时可能突破或反转。

四、突破测试（breakout test）  
空头趋势若跌破区间底，观察价格是否回拉测试突破点再下跌；若出现反转，做空。  
部分多头在区间底买入被跌破后加码 1–2 倍，盼反弹平仓；加码本身推高回拉，形成突破测试。

五、常见支撑 / 阻力来源  
• 趋势线、通道顶部  
• 前高、前低  
• 测量移动目标位  
• 各级别移动平均线；日线 50/100/150/200 日均  
• 大整数关口与期权执行价  
• 50 % 回撤：多头趋势中买点支撑，空头趋势中卖点阻力  
• 入场 / 加仓 / 平仓价，盈利目标位  
• 每日及昨日高、低、开、收；日线、周线、月线对应价位  
• 摆动区最高收盘与最低收盘  
• 趋势终点前的反转测试点  
任何一种都可能成为图表中最关键的支撑或阻力，取决于当时的上下文和强度。

结论  
支撑与阻力是市场波动背后的主导力量，也是交易机会的源泉。你得知道它们在哪，学会观察形态，理解 80 % 规则，留意突破测试，而不是被新闻牵着走。

价格行为学：二元决策
==========

原创 victorjia victorjia [价格行为学解析](javascript:void\(0\);)

在小说阅读器中沉浸阅读

我跟AI Brooks学习价格行为学2年了，今天是我写的第195篇文章，你也应该一起学习价格行为学。

Formulate your process in a binary decision tree so that each node has only two choices.

二叉决策树中的每个节点只有两种可能的选择。

人在复杂环境里，常常被犹豫拖住。要解决这个问题，最好的办法是二元决策：每一步都只能有两个结果。  
  

如果一个问题不能二元化，就说明你还没想清楚。二元迫使你做出判断：要么是，要么不是。模糊的地方，就是出错的地方。  
  

很多问题很复杂，没法一步到位。这时就要分层，把复杂的问题拆开，一层层归到二元。倒推是一种检验：如果某一层不能还原为二元，就说明思路里还有空白。  
  

真正的简单，不是少想，而是每个节点都能二元化。当系统形成闭环，就不会慌乱。因为你知道，无论遇到什么，都能落到“是”还是“否”。  
  

计算机的世界只有 0 和 1，却能支撑起整个文明。如果不是二进制，错误率会无限放大。  
  

复杂系统之所以混乱，是因为超过了二元。成功的路径都很相似，但失败的方式无数。你执行不下去，往往不是市场太复杂，而是系统里有些条件没有二元化。

  
不要搞复杂，并不是拒绝新元素，而是新元素必须能二元归类。比如用均线，就必须定义清楚，模糊的描述，只会带来混乱。

真正的系统，就是一套能落实到二元决策的方法。

构建二元的难点在于：把模糊的知识磨到可以落地。  
  

古人说：“一生二，二生三，三生万物。”二，是秩序；三，是混沌。这和“三体问题”一样：两个星体可预测，三个就进入混乱。

所以，二是边界，三是混沌。要在复杂中保持稳定，就必须把边界压回到二。
